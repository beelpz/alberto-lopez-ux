Visual Communication Overhaul for LATAM Players

Challenge

Free Fire faced critical visual inconsistency issues. The Revenue team created new Look & Feel designs weekly while campaigns ran for 1-2 months, causing confusion between lobby, shop, and promotional banners. LATAM players strongly prefer visual communication over text.

Research Process

  • Analyzed CTR data and heatmaps to identify interaction patterns
  • Conducted in-game surveys for initial feedback
  • Created mockups with marketing and design teams
  • Ran FGDs with 24 players across 4 segments (paying hardcore, active F2P, new users, non-players)
  • Monitored real-time interactions through behavioral observation

Key Insights

  • LATAM players strongly prefer visual communication over text
  • Visual inconsistency between game areas caused confusion
  • Clear, cohesive L&F significantly improves navigation
  • Campaign visibility improved with aligned visual elements

Solution

Implemented comprehensive visual communication strategy with drastically reduced text, unified Look & Feel system across all touchpoints, and design guidelines ensuring consistency while allowing campaign creativity.

Physical PPC Card Redesign

Challenge

Physical prepaid cards for in-game currency needed optimization to improve conversion rates, brand visibility, and value perception in retail environments.

Research Process

  • Created 6 different card design variants
  • Tested with current PPC users, non-users, and players using other payment methods
  • Optimized colors, character selection, information hierarchy, and layout
  • Iteratively refined based on multi-segment feedback

Key Insights

  • Character prominence significantly impacted brand recognition
  • Color choices affected perceived value and game association
  • Information hierarchy was crucial for retail visibility
  • Physical card design influenced digital payment method preferences

Solution

Developed optimized card design with prominent character artwork, strategic color usage, clear value proposition, and improved information hierarchy balancing shelf appeal with brand consistency.

Comprehensive In-Game Banner System

Challenge

Free Fire needed a unified banner system for diverse content including events, offers, features, ranked seasons, patches, and collaborations without disrupting gameplay flow.

Research Process

  • Catalogued all banner types and purposes
  • Mapped player journey from splash to gameplay
  • Conducted heatmap studies for optimal placement
  • Validated with focus groups
  • Tracked CTR, participation rates, DAU/MAU, and bounce rates

Solution

Created comprehensive banner ecosystem with unified visual language, minimal text approach, and strategic placement hierarchy. Implemented flexible templates for quick updates while maintaining design integrity.

Case Study Title

Context & Challenge

Players were struggling to discover new features in our mobile RPG game. Analytics showed that 65% of players never accessed the crafting system, despite it being a core gameplay mechanic. The challenge was to redesign the interface to improve feature discoverability without disrupting the existing user flow.

Research Process

1. Analytics Review: Analyzed user flow data to identify drop-off points
2. User Interviews: Conducted 15 interviews with active players
3. Usability Testing: Tested 3 design concepts with 20 participants
4. Heatmap Analysis: Tracked touch patterns across different UI layouts
5. A/B Testing: Rolled out changes to 10% of users for validation

Key Insights

  • Players expected crafting to be accessible from the inventory screen
  • The current icon was too similar to other menu items
  • New players needed better onboarding for complex features
  • Visual hierarchy didn't reflect feature importance

Design Solution

Created a new navigation system with clear visual hierarchy, contextual tooltips, and progressive disclosure. Added crafting shortcut to inventory screen and redesigned icons with distinct shapes and colors. Implemented a subtle animation to draw attention to new features during first use.
